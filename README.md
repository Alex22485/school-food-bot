# Telegram Bot "Школьная Столовая"

Бот для заказа блюд в школьной столовой.

## Установка и запуск

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Скопируйте `.env.example` в `.env` и добавьте ваш BOT_TOKEN
4. Запустите в режиме разработки: `npm run dev`
5. Запустите в production: `npm start`

## Безопасность

- ✅ Экранирование пользовательского ввода (защита от XSS)
- ✅ Rate limiting для защиты от спама
- ✅ Логирование всех действий
- ✅ Обработка ошибок
- ✅ Безопасное хранение токенов в .env

## Структура проекта

- `/src/bot` - логика бота
- `/src/utils` - утилиты (логирование и т.д.)
- `/src/middlewares` - мидлвары (безопасность, обработка ошибок)
- `/src/constants` - константы
- `/logs` - файлы логов
- `/backups` - бэкапы базы данных
